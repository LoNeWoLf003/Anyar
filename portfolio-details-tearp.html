Development of TeaRP

Main focus of the software is to reduce human dependency in getting facts and figures which would help the management to analyze the operations. The system also sets priority on making it a platform independent, secured, fast, robust and error free.

Hence the system is developed with following technical features:

Browser Based: The software is a browser based system which can be opened through any web browser like Internet Explorer 8, Google Chrome or Mozilla Firefox. The software can be run by just typing the URL in the browser from any machine which is in the server network. Hence it avoids installation of the software in each and every machine.

Data Integration:The Software can be used either online with central server at head office, or with a local server at each garden updating the central server at end of the day.Integrated software at head office will analyze the performance of each garden as well as a cumulative performance of all the gardens.

SMS and Mail alerts:With help of sms and mail pack, the software will have the capability of sending sms/mail to the mgmt. as soon as some deviation from normal parameter is noticed in the software.

Separate Web Interface:A separate web interface can be provided to view necessary reports from any place. At regular intervals data would be directly uploaded to web server.

Architecture:

  • Front End : Asp.Net
  • Back-End : MS-SQL Server
  • Reports : pdf and Excel

Kamjari: Kamjari is the major part of the module which tracks the attendance of all labors. The labors may be deployed either at fields for plucking or for factory or other works. On basis of their work, the wage is allocated to a specific account. The kamjari entry is done on basis of their khata no. A details of section or area of deployment is also maintained. There can be different method of calculation of their daily wage like thika or Hazira. The software is designed in a manner that this change of wage type and their slab details can be defined in the software. Contract labour and Cash plucking details can also be maintained in this system.Data from Electronic Weigh Machine The software has the ability to collect data from the electronic weighing machines/Face detection/Bio-Metric app. The kamjari is automatically generated from this data and it need not be entered into the software again.

Leave Management:The leave management of the labor is done automatically as per their entitlement. Various leaves have diff. way of treatment. Some are paid leave, some are partially paid leave while some are unpaid leave.

Wage and Salary Generation:At end of each week/fortnight/month, the wage is automatically generated from the Kamjari. An incentive is calculated as per the Govt. rules. All statutory data like extra earnings/deductions pre-defined in labour master takes effect during wage calculation. P.F., Ration etc. is automatically calculated during wage generation. P.F. Advance deduction and other deduction is also done and automatically posted to respective ledgers. Apart from the wage calculation, the Bonus, leave pay and arrear calculation with their effect on PF and RPF is also done through the software. The attendance, leave management, salary generation, leaves pay and bonus calculation of staff and sub-staff is also a key feature of the software. The attendance, leave management, bonus calculation and leave calculation can also be done through this system.

Manufacturing: All the manufacturing done in the factory as per :

  • Orthodox / CTC
  • Grade
  • Own leaf
  • Bought Leaf
  • Sorting
  • Packing
  • Daily Outturn
Several MIS can be generated from this data with each and every combination, along with reference to previous accounting year production which can benefit the Management for their business analysis.

Stores and Inventory: Store is almost the heart of total system. The features include:

  • Supplier Master
  • Supplier Approval
  • Indent
  • Purchase Order
  • GRN
  • Bill Confirmation
  • Local Purchase
  • Issue
  • Store Analysis Report
  • Stock Ledger

Power and Fuel Management The system keeps a detail as :

  • Consumption of Petrol/ Deisel / Gas
  • Detail of point of consumption like withering, drying, domestic etc.
  • Details of reflection of power to production
  • Electricity consumption from Electricity board with their point of consumption detail.
